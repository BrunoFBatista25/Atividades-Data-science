# Atividades-Data-science
Durante o Percurso do Semestre nos foi requerido duas Atividades, que buscaram tratar de forma particular diferente a mesma base de dados.

Atividade 1.  
A atividade 1 tinha por intuitos inicias nos familiarizar com a biblioteca Pandas em python, e indagava as seguintes questões:  
1º Quantas linhas e Colunas a base de dados possuia.  
2º Quais tipos de bases de Dados estavamos lidando.  
3º E a validação se havia algum valor nulo na base de dados.

Atividade 2.  
A atividade 2 teve como objetivo de desenvolver gráficos com a biblioteca mathplotlib por meio de histogramas, as indagações consistiam em:  
 
1º Quais as caracteristicas financeiras com o objetivo de montar sua lista.  
2º Observar os dados estatisticos e justifica-los.  
3º Tratar da visualização dos dados das Faturas (tipos de Dados Financeiros separados na 1º questão).  
4º A validação da análise dos Gráficos da Fatura.  
5º A visualização dos Gráficos de dados de Pagamentos (Outra lista).  
6º Adição de uma mascara Boleana em ambas listas.  
7º Ambas os Dados deveriam ser postos em um histograma com base logaritmica com Base 10 com Numpy.
